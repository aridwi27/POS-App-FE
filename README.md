# POS App

This is an application for restaurant cashiers that features add menu items, stores transaction history, and calculates menu totals at checkout. Built using the vue js framework and node js

# Dependencies

1. Axios
2. Bootstrap-vue
3. Vue-sweetalert2
4. vuex
